# Azus Cloner - Lynx
 [+] **Discord Server Cloner (Selfbot)**
---

## > Features
- Copies server name and icon
- Automatically deletes all channels, roles, and emojis in the destination server
- Clones roles with permissions, colors, and settings
- Clones categories, text channels, and voice channels (preserving structure)
- Clones emojis (including animated ones)

## > Requirements
- Python 3.8+
- Required libs:
- 
  ```bash
  pip install requests colorama
  ```
